# BC - Ruby Test

## Requirements
- Ruby 2.x or higher
- rspec gem


## Clone and run

1. Do `git clone https://github.com/elizar/bc-test.git && cd bc-test`
2. And then `gem install rspec`
3. Run the test with `rspec`
